**src/app/(auth)/login/page.tsx** — exact spec from 01-UI-SPEC.md:

This is a Client Component ('use client'). Use React Hook Form + Zod. Key requirements from the UI spec:
- Email field: autofocus on mount
- Password field: show/hide toggle via icon button with aria-label="Show password" / "Hide password"
- Validation fires on submit (not on blur) — set `mode: 'onSubmit'` in useForm
- Error display: above the submit button, NOT inline with fields (for auth errors from Supabase)
- Submit button: "Sign In" default, "Signing in..." when loading, disabled during submission
- On success: redirect to /dashboard/organizations (not /)
- On success: call router.refresh() to force Server Components to see new session
- Card: max-width 400px, bg-card, rounded-lg, p-8
- Product name: "VoiceOps" at 28px semibold (Display typography from spec)
- Tagline: "Voice AI Operations Platform" at 14px regular

Error messages from copywriting contract:
- Invalid credentials: "Invalid email or password. Check your credentials and try again."
- Network error: "Unable to connect. Check your internet connection and try again."
- Account disabled: "This account has been disabled. Contact your administrator."

Map Supabase error codes to these messages:
- error.message includes "Invalid login credentials" → use invalid credentials copy
- error.message includes "Network" or fetch error → use network copy
- Any other error → use the actual error.message as fallback

No "Forgot password" link — out of scope per requirements.

Color: The "Sign In" button must use the primary (indigo-500) variant — this is one of 6 reserved accent uses per the spec.

Responsive: card is full-width on mobile (w-full max-w-none below 480px), max-w-[400px] on desktop.

**src/app/(dashboard)/layout.tsx** — server component, reads user for sidebar:
```typescript
import { SidebarProvider, SidebarInset } from '@/components/ui/sidebar'
import { AppSidebar } from '@/components/layout/app-sidebar'
import { createClient } from '@/lib/supabase/server'
import { redirect } from 'next/navigation'

export default async function DashboardLayout({ children }: { children: React.ReactNode }) {
  const supabase = await createClient()
  const { data: { user } } = await supabase.auth.getUser()
  if (!user) redirect('/login')  // Belt-and-suspenders beyond middleware

  return (
    <SidebarProvider>
      <AppSidebar user={user} />
      <SidebarInset>
        {children}
      </SidebarInset>
    </SidebarProvider>
  )
}
```

**src/app/error.tsx** — root error boundary:

'use client'
import { useEffect } from 'react'
import { Button } from '@/components/ui/button'
import { useRouter } from 'next/navigation'

export default function Error({ error, reset }: { error: Error & { digest?: string }; reset: () => void }) {
  useEffect(() => {
    console.error(error)

  const router = useRouter()

  return (
    <div className="min-h-screen flex flex-col items-center justify-center gap-4">

      </Button>
    </div>

}
