```
npm install --save-dev vitest @vitejs/plugin-react
```

Create vitest.config.ts with this exact content:
```typescript
import { defineConfig } from 'vitest/config'
import react from '@vitejs/plugin-react'
import path from 'path'

export default defineConfig({
  plugins: [react()],
  test: {
    environment: 'node',
    globals: true,
    include: ['tests/**/*.test.ts', 'tests/**/*.test.tsx'],
    reporters: ['verbose'],
    testTimeout: 10000,
  },
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src'),
    },
  },
})
```

Add to package.json scripts section:
```json
"test": "vitest run",
"test:watch": "vitest"
```

Note: `environment: 'node'` is correct for integration tests against Supabase server clients. No jsdom needed for Wave 0 stubs — UI component tests are deferred to Phase 3.
